Cloud

Cloud computing is the on-demand availability of computer system resources, especially data storage and computing power, without direct active management by the user. The term is generally used to describe data centers available to many users over the Internet.

Module 2 - Big Data, Open Source Software and the Cloud

Web Servers and Web Sites

The story of IoT is one of devices that gather data and connect to servers that aggregate that data into powerful data stores. We trace the rise of the Browser as a data integrator that allows us to scale our systems across the world.
At the same time we trace the effect of Open Source Software as a driver of rapid innovation. In particular we look at Node.js, JavaScript and NPM (Node Package Manager) in allowing companies to build powerful web platforms.
In our exercises we look the importance of Web Servers, URLs and Routes that allow us to connect to and control devices across the world. Using these we can control a camera in our home or in Hong Kong with the same ease - physical distance being no problem.
